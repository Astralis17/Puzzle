The Polybius Square

Have you ever heard of this? probably not (unless I've rambled about it at some point) but that's ok, I'm here to explain now

Ok so the Polybius Square is thought to be made by an ancient greek philosopher called Polybius (shocker) and it works with a grid that is filled with the alphabet which is displayed below

And we encode information by taking the co-ordinates of the letter box, starting with row and then column, so 23 is second row f g h i/j k and column, aka "H"
You may notice that "i" and "j" take up a single box this is because if ever it's referenced it can be either of the two though typically it will be "i"

Ok so using the above chart/grid with the row first and column second we can encode any letter from the latin alphabet into a set of co-ordinates for this grid, this does however have it's downsides,
It removes capitalisation, spaces and numbers, this is because those specific characters are not in the table it's not possible to encode them.
There are other similar ciphers which include numbers but still not spaces .

Some people do however use 00 as a replacement for space
