The Ottendorf Cipher

The Ottendorf Cipher (also known as the Book Cipher) is a cipher that uses a piece of text (typically a book) as a key and a set of numbers to reference words or letters from said key text.

The formatting is as follows, Page:Line:Word (and then sometimes Letter)
For example, the code 17:9:3 references the 3rd word on the 9th line of the 17th page
Now for actual application, we'll use this image

Ok so this code 3:14:1, 0:7:1, 3:4:7, 3:7:5, 0:10:1
3:14:1; "It was only when she knelt to sponge up the mess,"
0:7:1; "is worth ten of the sword."
3:4:7; "Everything about today's delivery from the postman"
3:7:5; "even her age was wrong. And the bright orange socks"
0:10:1; "Book 1 Chapter 25"

And that all together gives us

"It is the wrong Book"


The Ottendorf Cipher can be used in a number of other ways, such as with lyrics of a song where the first number is line and second number is word Line:Word or maybe a playlist with titles, the possibilities are endless
