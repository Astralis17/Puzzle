The Ottendorf Cipher

The Ottendorf Cipher (also known as the Book Cipher) is a cipher that uses a piece of text (typically a book) as a key and a set of numbers to reference words or letters from said key text.

The formatting is as follows, Page:Line:Word (and then sometimes Letter)
For example, the code 17:9:3 references the 3rd word on the 9th line of the 17th page
